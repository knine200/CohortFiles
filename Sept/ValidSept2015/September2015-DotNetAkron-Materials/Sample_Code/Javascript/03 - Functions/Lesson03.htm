Lesson 03 - Declaring Functions

Functions in JavaScript can be assigned to variables and then called with 0..N parameters.

A function definition uses braces to capture logic relevant to the function and can optionally return a value to the caller

var functionName = function (param1, param2, paramN) { };

  1. var isEven = function (number) {
  2. var val = number % 2;
  3. return val == 0;
  4. };
