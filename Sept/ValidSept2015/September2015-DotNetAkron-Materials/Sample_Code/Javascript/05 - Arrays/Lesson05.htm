Lesson 05 - Arrays

Arrays are declared like other variables, but they contain their data in square braces []

Arrays are useful for storing lists of data. They maintain their order and the data within them can be accessed via index which starts at zero. Because JavaScript is a dynamically typed language arrays can contain mixed types.

Here are some examples of declaring arrays

Strings

  1. var myColors= ["blue", "red", "green", "brown"];

Numbers

  1. var myNumbers = [5, 10, 15, 20];

As mentioned earlier we can mix types

  1. var myMixedArray = [5, "something", true, "orange"];

Arrays in For Loops

Using our old friend length, we can also loop through an array with a for loop

  1. for (var i = 0; i < myColors.length; i++) {
  2. console.log("The color " + myColors[i] + " is at position " + i);
  3. }

push/pop

The push command pushes a new item to the end of an array. The pop method removes an item from the end of an array

  1. var letters = [];
  2. letters.push("A");
  3. letters.push("B", "C");
  4. console.log(letters);
  5. console.log(letters.pop());
  6. console.log(letters);

shift/unshift

shift and unshift are a similar concept to push and pop, but they work with the beginning of the array rather than the end

  1. var letters2 = [];
  2. letters2.shift("A");
  3. letters2.shift("B", "C");
  4. console.log(letters2);
  5. console.log(letters2.unshift());
  6. console.log(letters2);
