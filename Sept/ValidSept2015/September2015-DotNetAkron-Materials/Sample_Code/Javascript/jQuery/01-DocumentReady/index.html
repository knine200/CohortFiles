Lesson 01 - Document.Ready()

There are several ways to include jQuery into your project. The most common are to either download the latest script file from the jQuery site or use a Content-Delivery-Network (CDN) and let someone else host it for you.

Reference Downloaded Script

  1. <head>
  2. <script src="jquery-1.9.1.min.js">
  3. </head>

Reference CDN

  1. <head>
  2. <script src="//ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js">
  3. </head>

The (document).ready() event can be registered with a function to execute when the page is finished loading. It is a common entry point for setting up your jQuery events.

  1. $(document).ready(function() {
  2. alert('The document is ready!');
  3. });
