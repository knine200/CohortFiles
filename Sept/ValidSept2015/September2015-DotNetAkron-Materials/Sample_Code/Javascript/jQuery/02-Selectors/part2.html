Lesson 02 - Selectors Part 2

Let's do something fun! Here I've created a text input that has some html in it that is rendered on the other side of the screen. You can type any selector into the input below, click the button, and it will highlight the elements selected.

  1. Give a tag name like 'p' to select all elements of a tag
  2. Use a hash # to select an element with a specific id
  3. Use a period . to select elements with a certain css class
  4. Colons : are special selectors, try li:last-child for example
  5. You can combine selectors like div.rectangle to only select divs with the class rectangle
  6. Use a comma to combine selectors, for example p.intro,li would select all the li tags and the paragraph element with the intro class
