Lesson 02 - Selectors

Before you can manipulate a DOM element, you have to find it! Thus jQuery has built in support for selectors. The selector syntax starts with the jQuery $ sign and in parenthesis builds an expression to indicate which items you want to select for manipulation.

For example, here we have a hyperlink that will open the jQuery site into a new browser tab/window

jQuery Site

In the referenced script, I have selected all hyperlinks and will pop a message when the user clicks it like so:

  1. $(document).ready(function() {
  2. $('a').click(function( event ) {
  3. alert( "Thanks for visiting!" );
  4. });
  5. });
