Lesson 01 - Variables

Declare variables by using the var keyword

JavaScript has strings:

  1. var myString = "Hello World";

JavaScript has numbers:

  1. var myFirstNumber = 1;

Which by the way are more like doubles, notice it doesn't differentiate between ints and decimals:

  1. var myFirstNumber = 1.5;

JavaScript has booleans too:

  1. var myBool = true;

JavaScript is a dynamic-typed language, thus I can just go ahead and put a number in myString:

  1. myString = 5;

Use the developer tools built into the browser to check the console by hitting F12
