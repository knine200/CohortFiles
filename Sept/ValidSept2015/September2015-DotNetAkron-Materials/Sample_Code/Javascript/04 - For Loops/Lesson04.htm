Lesson 04 - For Loops

For loops in JavaScript are almost exactly the same as in C#. For any for loop you must identify a loop counter, control condition, and increment amount:

  1. for (var counter = 1; counter < 11; counter++) {
  2. console.log(counter);
  3. }
  4.  
  5. for (var i = 5; i <= 50; i += 5) {
  6. console.log(i);
  7. }
